 Projeto: Combate ao Bullying

Este projeto foi desenvolvido com o objetivo de conscientizar as pessoas sobre o bullying, apresentando informações importantes de forma simples, clara e interativa. O site busca incentivar o respeito, a empatia e a convivência saudável entre as pessoas, mostrando que atitudes de violência física, verbal, psicológica ou virtual podem causar sérios impactos na vida das vítimas.

O desenvolvimento do projeto utilizou as tecnologias HTML, CSS e JavaScript. O HTML foi responsável por estruturar o conteúdo da página, organizando as seções e os textos informativos. O CSS foi utilizado para criar um visual moderno, agradável e responsivo, tornando a navegação mais fácil e atraente. Já o JavaScript adicionou interatividade ao site por meio de um botão que exibe uma mensagem de conscientização, incentivando os visitantes a refletirem sobre a importância do respeito e da solidariedade.

Além de explicar o que é o bullying, o site apresenta seus principais tipos, como o bullying físico, verbal, psicológico, social e o cyberbullying, além de oferecer orientações sobre como prevenir esse problema. O projeto também destaca a importância de denunciar situações de bullying e de buscar ajuda de familiares, professores ou outros adultos de confiança quando necessário.

O principal objetivo deste trabalho é contribuir para a construção de um ambiente mais seguro, acolhedor e respeitoso, promovendo a conscientização de que pequenas atitudes de empatia e respeito podem fazer uma grande diferença na vida das pessoas.
